ProjectGasu is a real-time live data air quality monitor created by 4 sophomores from Troy High School in Fullerton, California. Paired with a Raspberry Pi and a sensor, the system can detect abnormalities in PM2.5 and PM10 concentrations. Utilized Figma for frontend design, python for backend, SOLIDWORKS for case design, and an SDS011 air quality sensor. Created by Terence Ren, Rohan Chakrabortygupta, Cristian Liu, and Ryan Jian.
